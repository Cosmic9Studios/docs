<AccordionGroup>
    <Accordion title="How do I gain boosts?" icon="fire">
        Only managers can gain boosts. If you are a manager and you need more boosts, you can request more from the Vork admin or wait for them to automatically reset.
    </Accordion>

    <Accordion title="Why don't I see any connections?" icon="plug">
        If you don't see any connections, it's likely that your admin hasn't enabled them. Reach out to your admin to enable connections.
    </Accordion>

    <Accordion title="How do I request a new Connections" icon="plug">
        If you're an admin and you need a new connection, you can request one at: [support@vork.app](mailto:support@vork.app)
    </Accordion>

    <Accordion title="How do I request new Automations?" icon="gear">
        If you're an admin and you would like a new automation, you can request one at: [support@vork.app](mailto:support@vork.app)
    </Accordion>
</AccordionGroup>
